📁 VIDEOS FOLDER — Shraddha Institute
======================================

OPTION A — YouTube (Recommended)
──────────────────────────────────
Upload your videos to YouTube (public or unlisted) and paste the YouTube
video ID into the code. The website will embed it directly.

OPTION B — Local MP4 file
──────────────────────────────────
DROP VIDEO FILES HERE with EXACTLY these file names:

intro.mp4         → 60-90 second intro / promo video about Shraddha Institute
testimonial-1.mp4 → Video testimonial of a teacher (30-60 seconds)
exam-day.mp4      → Exam / competition event video
prize-ceremony.mp4 → Prize / certificate distribution event

RECOMMENDED:
  - Resolution: 720p (1280x720) — good balance of quality & speed
  - Format: MP4 (H.264 codec)
  - Duration: max 2 minutes per video for fast loading
  - File size: max 20MB per video

HOW TO COMPRESS:
  Use https://www.handbrake.fr (free) to compress large videos
